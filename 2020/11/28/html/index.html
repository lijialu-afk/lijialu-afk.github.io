html

HTML 标题
HTML 标题（Heading）是通过 h1 标签来定义的。

HTML 段落
HTML 段落是通过标签 p 来定义的。

HTML 链接
HTML 链接是通过标签 a 来定义的。

提示:在 href=链接的地址，后面的文字是代替链接的文字。
HTML 图像
HTML 图像是通过标签 img 来定义的.

提示： 图像的名称和尺寸是以属性的形式提供的。
定义图像的语法是：
《img src=”url” /》（使用英文字符，url可以是路径也可以是链接）
具体内容详见（https://www.w3school.com.cn/html/html_images.asp）

插入本地图片方法

绝对路径
比如，你的图片存放在D盘images目录里，名称是img.png。那么绝对路径应该是这样：D:\images\img.png。
相对路径
如果，你的html文件存放的位置跟图片一样，那么就直接表达：img.png。
子路径
那如果，你的html文件存放在D盘里，那么就是子路径了：images/img.png。
父路径
html文件存放在D盘里的images文件里的css文件中，那么就使用父路径了：../img.png，当然了，也可以../../images/img.png。
